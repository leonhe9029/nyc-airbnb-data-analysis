## NYC Airbnb Data Analysis

Analyzed 48,000+ NYC Airbnb listings to explore pricing patterns and borough-level differences.

**Tools:** Python, pandas, matplotlib  
**Techniques:** Data cleaning, exploratory data analysis, grouping, visualization  

**Key Findings:**
- Airbnb prices are right-skewed, with most listings concentrated at lower price points
- Manhattan has the highest average listing prices compared to other boroughs
- Removing extreme outliers improves interpretability of pricing trends
